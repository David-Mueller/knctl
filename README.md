# Knative CLI

Knative CLI (`knctl`) provides simple set of commands to interact with a Knative installation.

If you are interested in contributing, see [CONTRIBUTING.md](./CONTRIBUTING.md).

## Build

```bash
# export GOPATH=...

$ ./hack/build.sh

$ ./knctl version
```
